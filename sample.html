Intro to Programming Nanodegree

Setting Up

You’ll be using HyperText Markup Language to create web pages. Two necessary items: a place to save your code (i.e. a text editor), and a web browser to view the html files.

Building your first Webpage

In order to save your code as a html file and your browser to be able to interpret the html you must save your file with the appropriate extension of .html That’s one of the main reasons we use a text editor rather than a word processor it allows us to save our code with the proper extensions.

In order to view the rendered webpage in a web browser such as Firefox, Safari or Chrome you must do the following:

In Mac use Finder to reach the file location. Once you find the file you want to use double-click it and it should open in a web browser.

In Safari you can also use File- Open File and select the desired location to open your html code.

To make changes to the code you must first edit within your text editor and then save and refresh your browser.

  • Procedural Thinking - involves creating clear and perfectly unambiguous instructions for a computer to follow.

  • Abstract Thinking - finding generality amongst seemingly different things. Abstraction helps programmers avoid unnecessary repetition of work.

  • Systems Thinking- Breaking a big problem down to smaller pieces.

The basics of the Web

HTML is HyperText Markup Language

The web is comprised of hyperlinks.

The web was invented in the 1990s there are an estimated 30 billion pages

Types of files that can be found on the Web:

  • Plain text
  • HTML
  • Images
  • Videos
  • Music

Components of the Web

The major pieces are you, your computer running a web browser, the internet, and servers.

Your browser makes requests (using a protocol called HTTP) via the Internet to servers. The servers communicate with your browser and display files. Some examples of browsers include:

  • Internet Explorer

  • Chrome

  • Safari

  • Firefox

The servers are computers just like yours except they are optimized for sitting in a closet and hosting files. The HTTP protocol is a basic protocol that your browser uses to communicate with servers.

HTML Basics

What is HTML?

HTML is made up of:

  • text content- what you see
  • markup- what it look likes
  • references to other documents- i.e images and videos
  • links to other pages
